# Data Directory

Store your promoter sequence data here:

## File Types:
- `*.fasta` - Promoter sequences in FASTA format
- `*.csv` - Tabular data with promoter info and experimental results
- `*.txt` - Raw sequence files

## Expected Files:
- `c_acnes_promoters.fasta` - Your C. acnes strong promoter sequences
- `p3_promoter.fasta` - The weak P3 promoter from S. aureus
- `experimental_data.csv` - Any GFP expression or strength data you have

## Format Example:
```
>Promoter_1_Strong
TTGACAATGATTATCGCGATCGATTTATAAT...
>Promoter_2_Medium
TTGAAAATGATTATCGCGATCGATTTATAAT...
```
