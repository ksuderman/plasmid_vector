# Scripts Directory

Python scripts for promoter analysis:

## Main Analysis Scripts:
- `sequence_analyzer.py` - Basic sequence analysis and feature extraction
- `strength_predictor.py` - ML models for promoter strength prediction
- `motif_optimizer.py` - -10/-35 region optimization tools
- `data_preprocessor.py` - Clean and format input data

## Utility Scripts:
- `utils.py` - Common functions for sequence handling
- `visualizations.py` - Plotting and visualization tools
